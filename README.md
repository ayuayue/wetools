# WeTools - 开发者工具箱

[![Flutter Version](https://img.shields.io/badge/Flutter-3.6.0-blue.svg)](https://flutter.dev/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

一个使用 Flutter 开发的轻量级开发者工具箱，提供常用的编码解码、格式化、加密等功能。

## 功能特性

- **JWT 工具**
  - JWT Token 编码/解码
  - 支持自定义密钥
  - 实时预览结果

- **URL 工具**
  - URL 编码/解码
  - 支持中文字符
  - 复制结果功能

- **Base64 工具**
  - 文本编码/解码
  - 支持 Unicode 字符
  - 错误提示功能

- **JSON 工具**
  - JSON 格式化
  - JSON 压缩
  - Unicode 转中文
  - 支持语法校验

- **Hash 工具**
  - MD5 计算
  - SHA1 计算
  - SHA256 计算
  - SM3 计算
  - 支持加盐处理
  - 支持大小写切换

- **文本工具**
  - 去除首尾空白
  - 大小写转换
  - 中英文标点转换
  - 字符统计（包含 UTF-8/GBK 编码长度）
